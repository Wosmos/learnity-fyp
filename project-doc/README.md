## ğŸ® Key Features

### **Gamification System**
- **Multi-type Streaks**: Learning, social, consistency streaks
- **XP & Leveling**: Dynamic multipliers and visual celebrations
- **Achievement Badges**: Milestone, skill, and collaboration badges
- **Social Competition**: Leaderboards and group challenges

### **Role-Based Architecture**
- **Students**: Book tutors, join study groups, watch content
- **Teachers**: Set pricing, upload videos, conduct sessions
- **Admins**: Review applications, manage users, view analytics

### **Technology Stack (Free Tools)**
- **Frontend**: Next.js 14 + Tailwind CSS + shadcn/ui
- **Database**: PostgreSQL (Neon) + Prisma ORM
- **Real-time**: Firebase Firestore + Storage
- **Video**: Jitsi Meet
- **Deployment**: Vercel + GitHub Actions
